# forum_comment_generator
Python Forum Comment Generator


*Project from my Intro to Python college course*

In this project I created a forum that allows you to add and view comments. The program pulls up a menu with options to display stored comments, add a comment with your name and a date, and to exit the program. Comments are stored in a file using JSON.
